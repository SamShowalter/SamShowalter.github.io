Data Science Analyst: Merchandising Insights

Data Science arm of Kroger stores. Developed and outlier detection system in SQL to programmatically parse incoming Controlled Store Test (CST) data for executional errors. CSTs allow Kroger and its consumer packaged goods (CPG) providers (P&G, Burt's Bees, etc.) to experiment with different store layouts and product packaging. These tests are difficult to conduct because holding extraneous variables constant is extremely difficult. My task was to validate testing data was suitable for further analysis and not greatly impacted by confounding factors. Ultimately, my solution ran in less than 30 seconds and detected over 85% of known executional issues historically, saving analysts 4 - 8 hours per test.

Purdue University

Chemistry Research Fellow

Summer undergraduate research fellow (SURF) in Dr. Suzanne Bart's inorganic Chemistry lab group. Mentored by PhD student John Kiernicki and worked to generate Rhodium and Iridium organo-metallic complexes by successfully attaching organic ligands to the structure. Metal-mediated organic transformations with f-block compounds, particularly those with low oxidation states, are difficult to conduct. Over the summer I was able to generate one Iridium crystal structure of adequate quality, and determine its resulting structure using H1 NMR (certified over summer), IR spectroscopy, and other diagnostic tests.

Selected Projects

Hunger Snakes: Neuroevolution of Fixed Topologies and Advsersarial Learning

There are many examples on the internet of developers and Data Scientists utilizing Neuroevolution of Augmenting Topologies to teach the agents of simple computer games to play (Snake, Mario Bros., etc.). However, few of these implementations conduct such an experiment under adversarial circumstances, when multiple agents compete for limited resources. In this project I build, from scratch, an adversarial implementation of Neuroevolution for the game snake. For simplicity, topologies were fixed for all agents based on a common fitness function.

Code Publication Demo

Web Application for Image Classification in the CalTech256 Image Dataset

The CalTech256 Image dataset (LINK HERE) was created to assess the performance of different image classification models, particular under different input data size (number of photos) constraints. Using a transfer learning model build on the Xception classifier, myself and small team I led were able to achieve a holistic macro and micro F1 Score of 86% across all classes. We then integrated this model into a frontend web application where a user can upload any photo and see the top 10 class prediction probabilities as well as the overall prediction.

Shakespeare Chains: Markov Chain Story Writing

Markov Chains are simple networks that rely on probablistic state transfer. These transfer probabilities are based on a specific corpus, something that can be unfluenced by the user. This project allows the user to upload a text file (PDF support coming shortly!) and then generate a unique story with the style of the input authors. This process will include all punctuation and spacing is provided by the system, allowing for stylistic spacing like in poetry to be accommodated.
